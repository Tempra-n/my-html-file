📚 シリーズ構成

第1巻：認知心理学 完全網羅編 - 記憶・注意・知覚・学習現象
第2巻：発達心理学 完全版 - 理論・実験・現象・応用

📖 本シリーズの使用方法

  • 段階的学習：第1巻→第2巻→第3巻→第4巻の順序で進める
  • 反復練習：各巻を最低2回は解答する
  • 時間管理：実際の試験時間を設定して取り組む
  • 論述力向上：解答は必ず文章で記述する
  • 弱点克服：間違えた問題は理論を再学習する
第1巻：認知心理学 完全網羅編
【第1回】記憶システム基礎編（120分）
問題1-1：記憶システムの階層構造 (15点)
アトキンソン＝シフリンの記憶モデルからバドリーのワーキングメモリモデルへの発展について説明しなさい。
  • 従来モデルの限界
  • 新モデルの革新性
  • 実験的証拠の変遷
  • 現在の評価と課題
問題1-2：ワーキングメモリの詳細分析 (20点)
バドリーのワーキングメモリの4構成要素について、以下を含めて詳述しなさい。
  1. 中央実行系：注意制御機能、容量限界、実験的証拠
  2. 音韻ループ：構造、音韻類似性効果、語長効果
  3. 視空間スケッチパッド：機能、心的回転との関係
  4. エピソード的バッファ：追加された背景、他システムとの統合
問題1-3：二重課題パラダイムの応用 (15点)
二重課題実験について、以下の実験を設計し、予想結果を説明しなさい。
  • 実験目的：音韻ループと視空間スケッチパッドの独立性検証
  • 実験条件：主課題・副課題の組み合わせ
  • 測定指標：パフォーマンス低下の程度
  • 予想結果：理論的根拠とともに
問題1-4：系列位置効果の多面的理解 (15点)
系列位置効果について、以下を統合的に説明しなさい。
  • 初頭効果と新近効果のメカニズム
  • 遅延再生による新近効果の消失
  • 干渉課題の影響
  • ワーキングメモリとの関係
【第2回】長期記憶システム編（120分）
問題2-1：宣言的記憶の二重構造 (20点)
エピソード記憶と意味記憶について、以下の観点から比較分析しなさい。
  1. 定義と特徴：タルヴィングの提唱内容
  2. 発達的観点：幼児期からの発達過程
  3. 神経基盤：海馬・側頭葉の役割分担
  4. 病理学的解離：認知症・健忘症での症状差
  5. 実験的検証：remember/know判断の実験パラダイム
問題2-2：手続き的記憶の特性 (15点)
手続き的記憶について、以下を含めて説明しなさい。
  • 定義と特徴：意識性、言語化可能性
  • 神経基盤：大脳基底核の役割
  • 学習特性：習得過程、忘却抵抗性
  • 日常例：運動技能、認知技能
  • 障害例：パーキンソン病、ハンチントン病
【第3回】注意メカニズム編（120分）
問題3-1：選択的注意の理論発展 (20点)
選択的注意理論の発展について、以下の流れで説明しなさい。
  1. ブロードベントのフィルター理論：初期選択説
  2. トライズマンの減衰理論：閾値概念の導入
  3. ドイッチェ＝ノーマンの後期選択説：意味分析の重要性
  4. 現代の統合的理解：柔軟な注意配分
問題3-2：注意の瞬きとその応用 (15点)
注意の瞬き（attentional blink）について説明し、以下の応用場面を分析しなさい。
  • 実験パラダイム：RSVP課題
  • 時間特性：200-500msの盲点
  • 理論的解釈：注意資源の枯渇
  • 応用場面：運転時の見落とし、医療現場での見逃し

第3巻：統合型・ケーススタディ編
第1部：統合型問題
【問題1】臨床心理学・異常心理学統合問題 (25点)
状況： 25歳の会社員Aさんが「最近2ヶ月間、気分が沈み、何をしても楽しくない。夜中に何度も目が覚め、朝は特に調子が悪い。仕事でのミスが増え、上司から注意されることが多くなった」と相談に来ました。
課題：
  1. DSM-5の診断基準に基づき、最も考えられる診断名とその根拠を示しなさい（6点）
  2. 初回面接で実施すべきアセスメント手法を3つ挙げ、それぞれの目的を説明しなさい（9点）
  3. 認知療法を適用する場合の治療構造と主要技法を説明しなさい（10点）
【問題2】学習心理学・教育心理学統合問題 (20点)
状況： 中学校の英語教師が「生徒たちが宿題をやってこない。注意しても一時的にしか改善されない」と相談しています。
課題：
  1. この問題をオペラント条件づけの三項随伴性で分析しなさい（5点）
  2. プレマックの原理を応用した具体的な介入方法を提案しなさい（5点）
  3. 自己決定理論の3つの基本的欲求の観点から問題を分析し、改善策を提示しなさい（5点）
  4. 学習性無力感を防ぐための教師の関わり方を説明しなさい（5点）
第2部：ケーススタディ問題
【ケース1】うつ病の心理療法 (25点)
クライエント情報：
・30歳女性、会社員
・主訴：「何をしても楽しくない、自分は価値のない人間だ」
・症状：抑うつ気分、興味・喜びの著しい減退、不眠、食欲低下、集中困難
・持続期間：3ヶ月
・生活史：完璧主義的性格、最近昇進したが責任の重さにプレッシャーを感じている
問い：
  1. このクライエントに対するケースフォーミュレーションを作成しなさい。生物学的・心理学的・社会的要因を含めること（10点）
  2. TAT（主題統覚検査）を実施する意義と、予想される反応パターンを説明しなさい（7点）
  3. 認知療法の共同実証主義に基づいて、クライエントの自動思考「自分は価値のない人間だ」をどう扱うか、具体的なセッションの進め方を示しなさい（8点）
【ケース2】発達障害児の包括的支援 (30点)
事例：
8歳男児、小学2年生。「授業に集中できない、友達とのトラブルが多い」と担任から相談。

詳細情報：
・症状：注意集中困難、多動性、衝動性
・学習面：読み書きの困難、計算はできる
・社会面：友達への攻撃行動、ルールの理解困難
・家庭：両親は困惑、しつけに悩んでいる
・発達歴：言語発達の遅れ、運動発達は正常
課題：
  1. 考えられる診断と鑑別診断のポイントを説明しなさい（10点）
  2. 包括的アセスメントの計画を立てなさい（10点）
  3. 学校・家庭・医療機関の連携による総合的支援計画を提案しなさい（10点）
