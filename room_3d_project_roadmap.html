📋 Phase 1: 部屋情報収集・計測（1-2日）
1
寸法計測
部屋の縦×横×高さを正確に測定。窓、ドア、コンセントの位置も記録
2
家具リスト作成
現在の家具の種類、サイズ、配置を詳細にリストアップ
3
写真撮影
複数アングルから部屋を撮影。3D化の参考資料として活用
4
改善要望整理
現在の不満点、理想の使い方、追加したい機能を明確化
🤖 Claude連携ポイント
「6畳の部屋にベッド、机、クローゼットを配置したい。現在は収納が不足している」のような自然な日本語で要望を伝えるだけで、Claudeが最適な計測項目と手順を提案します。
⏱️ 所要時間
1-2日（空いた時間で徐々に作業可能）
🔧 Phase 2: Claude連携システム構築（2-3日）
1
Blender環境準備
Blenderのインストールと基本設定。Python環境の確認
2
コード生成方式選択
初心者向けのコード生成支援から上級者向けAPI連携まで、レベルに応じて選択
3
動作テスト
「赤い立方体を作成」などの簡単な指示でシステムが正常動作することを確認
4
部屋専用プロンプト作成
部屋のモデリングに特化した効率的なプロンプトテンプレートを準備
🛠️ 使用技術スタック
Blender 4.0+
Python 3.10+
Claude API
bpy (Blender Python API)
bmesh
🤖 Claude連携ポイント
システム構築中にエラーが発生した場合も、エラーメッセージをClaudeに送るだけで詳細な解決策を提案。プログラミング初心者でも安心して進められます。
⏱️ 所要時間
2-3日（初心者の場合は5-7日を推奨）
🏗️ Phase 3: 部屋の基本構造3D化（3-5日）
1
部屋の箱作成
正確な寸法で部屋の基本構造（壁、床、天井）を作成
2
開口部の追加
窓、ドア、クローゼットなどの開口部を正確な位置に配置
3
基本マテリアル設定
壁紙、フローリング、天井の色や質感を実際の部屋に合わせて設定
4
照明の基本設定
窓からの自然光、室内照明を配置して、実際の明るさを再現
🤖 Claude連携ポイント
「6畳の部屋で、東向きの窓が2つあり、フローリングは明るい茶色です」と伝えるだけで、Claudeが適切なBlenderコードを生成。寸法や材質設定も自動で最適化されます。
⏱️ 所要時間
3-5日（段階的に進めて、毎日少しずつ改善）
🪑 Phase 4: 家具・オブジェクト配置（5-7日）
1
家具モデル作成/取得
現在の家具を3Dモデル化、または類似のフリー素材を取得
2
現状再現配置
実際の配置を忠実に3D空間で再現
3
動線チェック機能
歩行経路を可視化し、動線の問題点を発見
4
インタラクティブ操作
マウスで家具をドラッグ&ドロップできる仕組みを実装
🤖 Claude連携ポイント
「ベッドを窓際に移動して、机を部屋の中央寄りに配置してください。生活動線も確認したいです」のようなリクエストで、Claudeが家具移動のコードと動線分析を同時に実行します。
⏱️ 所要時間
5-7日（家具の数や複雑さによって調整）
✨ Phase 5: 最適化・改善提案（3-5日）
1
複数レイアウト案生成
Claudeと協力して、異なるコンセプトのレイアウト案を複数作成
2

3

4

⏱️ 所要時間
3-5日（理想を追求するほど時間をかけられる楽しい段階）
